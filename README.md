# souutsu-manage

躁鬱の状態をスマートウォッチで計測したデータをもとに評価するためのアプリ。  
多分まだ動かない。

## 概要

このプロジェクトは、双極性障害の当事者が自分の状態変化に早く気づくためのセルフモニタリング支援システムである。

睡眠、活動量、心拍、服薬、気分ログをもとに、本人の通常状態からの逸脱を検知し、Discord Bot を通じて説明可能な警告を返す。

目的は診断ではなく、当事者が自分の状態を判断するための補助輪を提供することである。

## 目的

- 双極性障害の当事者が、自分の状態変化に早く気づけるようにする
- 睡眠・活動量・心拍・服薬・気分ログを統合して確認できるようにする
- 本人の通常状態からの逸脱を検知する
- Discord Bot を通じて、説明可能な警告を返す
- 診断ではなく、自己判断を補助する仕組みを作る

## 注意

このアプリは医療的な診断を行うものではありません。  
双極性障害の診断・治療・服薬判断は、必ず医師や専門家に相談してください。
